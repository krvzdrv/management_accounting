# Включение Apps Script API для аккаунта

## Проблема

Авторизация прошла успешно, но нужно включить Apps Script API для вашего Google аккаунта.

## Решение: Включить Apps Script API

### Шаг 1: Откройте настройки Apps Script

Откройте эту ссылку в браузере:

```
https://script.google.com/home/usersettings
```

Или:
1. Откройте [script.google.com](https://script.google.com)
2. В меню слева нажмите **Settings** (Настройки) или **⚙️**
3. Перейдите в **Google Apps Script API**

### Шаг 2: Включите API

На странице настроек:

1. Найдите раздел **"Google Apps Script API"**
2. Включите переключатель (Toggle) или нажмите **"Enable"** (Включить)
3. Может появиться предупреждение - нажмите **"Enable"** или **"Allow"**

### Шаг 3: Подождите несколько минут

После включения API:
- Подождите 2-3 минуты
- Изменения могут применяться не сразу

### Шаг 4: Запустите скрипт снова

После включения API запустите скрипт снова:

```bash
cd "/Volumes/02 Data/work/Alumineu/GitHub/management_accounting"
node update-scripts.js
```

Теперь файлы должны обновиться успешно!

## Альтернативный способ: Через Google Cloud Console

Если не можете найти настройки в Apps Script:

1. Откройте [Google Cloud Console](https://console.cloud.google.com)
2. Выберите проект: **calcium-vial-378411**
3. Перейдите: **APIs & Services** → **Library**
4. Найдите **"Google Apps Script API"**
5. Убедитесь что API включен (статус "Enabled")

Но это может не помочь, так как нужно включить API именно для вашего аккаунта, а не только в проекте.
