# Инструкция по настройке Google Таблиц

## Структура файла Google Таблиц

Создайте один файл Google Таблиц с следующими листами (sheets):

### Листы справочников:
1. **Контрагенты** - справочник контрагентов
2. **Товары** - справочник товаров
3. **Валюты** - курсы валют
4. **Категории расходов** - категории расходов

### Листы операций:
5. **Закупки** - все закупки товаров
6. **Продажи** - все продажи товаров
7. **Расходы** - все расходы
8. **Движения денег** - все денежные переводы между компаниями

### Листы отчетов:
9. **Взаиморасчеты** - кто кому должен
10. **P&L** - отчет о прибылях и убытках
11. **Остатки** - складские остатки
12. **Статистика расходов** - анализ расходов
13. **Финансирование** - кто за что платил

## Порядок создания

1. Создайте новый файл Google Таблиц
2. Переименуйте лист "Лист1" в "Контрагенты"
3. Добавьте остальные листы через "+" внизу
4. Скопируйте структуру колонок из файлов structure_*.md
5. Добавьте формулы из structure_reports.md

## Рекомендации по оформлению

- Используйте закрепление первой строки (View → Freeze → 1 row)
- Применяйте условное форматирование для статусов
- Используйте фильтры для удобной работы с данными
- Создайте дашборд на отдельном листе с ключевыми показателями

## Защита данных

- Заблокируйте формулы в отчетах (правой кнопкой → Protect range)
- Оставьте возможность редактирования только в таблицах операций
- Используйте Data Validation для полей с выбором (ID контрагентов, товаров и т.д.)
