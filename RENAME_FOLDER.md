# Переименование папки проекта

## Инструкция по переименованию

### Шаг 1: Переименовать папку

**В Finder (Mac):**
1. Откройте Finder
2. Перейдите в `/Volumes/02 Data/work/Alumineu/GitHub/`
3. Найдите папку `management accounting`
4. Нажмите на название папки (или F2)
5. Переименуйте в `management_accounting`
6. Нажмите Enter

**Или через терминал:**
```bash
cd "/Volumes/02 Data/work/Alumineu/GitHub"
mv "management accounting" "management_accounting"
```

### Шаг 2: Обновить путь в терминале

После переименования перейдите в новую папку:
```bash
cd "/Volumes/02 Data/work/Alumineu/GitHub/management_accounting"
```

### Шаг 3: Проверка

Убедитесь что вы в правильной папке:
```bash
pwd
ls -la
```

Должны видеть файлы проекта (package.json, update-scripts.js и т.д.)
