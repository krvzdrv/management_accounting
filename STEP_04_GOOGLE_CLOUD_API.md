# ШАГ 4: Настройка Google Cloud - Включение Apps Script API

## Что нужно сделать

### 1. Откройте Google Cloud Console

Откройте в браузере: [console.cloud.google.com](https://console.cloud.google.com)

### 2. Выберите или создайте проект

**Если у вас уже есть проект:**
- Выберите его из выпадающего списка вверху

**Если проекта нет:**
- Нажмите на название проекта вверху
- Нажмите **"New Project"** (Новый проект)
- Введите название: "Management Accounting" (или любое другое)
- Нажмите **Create**

### 3. Включите Apps Script API

1. В меню слева выберите **APIs & Services** → **Library**
   (Или найдите в поиске: "APIs & Services")

2. В поиске вверху страницы введите: **"Google Apps Script API"**

3. Нажмите на результат **"Google Apps Script API"**

4. На странице API нажмите кнопку **"Enable"** (Включить)

5. Подождите несколько секунд - должно появиться сообщение что API включен

### 4. Проверка

- ✅ API должен быть включен (статус "Enabled")
- ✅ Вверху страницы должна быть зеленая галочка или статус "API enabled"
