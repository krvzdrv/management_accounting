# Структура таблиц операций

## 1. Закупки (Purchases)

| Поле | Тип | Описание | Пример |
|------|-----|----------|--------|
| Дата | Дата | Дата закупки | 2024-01-15 |
| Номер документа | Текст | Номер накладной/счета | INV-001 |
| Поставщик ID | Число | ID из справочника контрагентов | 3 (Flexy) |
| Товар ID | Число | ID из справочника товаров | 1 |
| Количество | Число | Количество товара | 100 |
| Цена за единицу | Число | Цена без НДС | 10.50 |
| Валюта | Текст | Валюта операции | EUR |
| Сумма без НДС | Число | = Количество × Цена | 1050.00 |
| НДС % | Число | Ставка НДС | 20 |
| НДС сумма | Число | = Сумма × НДС% | 210.00 |
| Сумма с НДС | Число | = Сумма + НДС | 1260.00 |
| Курс валюты | Число | Курс на дату операции | 1.0 |
| Сумма в EUR | Число | = Сумма × Курс | 1050.00 |
| Кто оплатил | Текст | Кто фактически оплатил | Версатрейд / Алюминею / Flexy |
| Статус оплаты | Текст | Оплачено / Не оплачено | Оплачено |
| Примечание | Текст | Дополнительная информация | - |

**Важно:** 
- Поле "Кто оплатил" критично для отслеживания финансирования операций
- Если оплатила другая компания, это создает задолженность
