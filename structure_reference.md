# Структура справочников

## 1. Контрагенты (Counterparties)

| Поле | Тип | Описание | Пример |
|------|-----|----------|--------|
| ID | Число | Уникальный идентификатор | 1 |
| Название | Текст | Название компании/клиента | Алюминею |
| Тип | Текст | Тип контрагента | Компания / Клиент / Поставщик |
| Валюта по умолчанию | Текст | Основная валюта | EUR |
| Примечание | Текст | Дополнительная информация | - |

**Примеры записей:**
- Алюминею (Компания, EUR)
- Версатрейд (Компания, EUR)
- Flexy (Поставщик, EUR)
- Посредник (Поставщик, EUR)
- Завод (Поставщик, EUR)
- Клиент 1, Клиент 2, и т.д.

## 2. Товары (Products)

| Поле | Тип | Описание | Пример |
|------|-----|----------|--------|
| ID | Число | Уникальный идентификатор | 1 |
| Артикул | Текст | Артикул товара | PROFILE-001 |
| Название | Текст | Название профиля | Профиль алюминиевый 20x20 |
| Единица измерения | Текст | Ед. изм. | м / шт / кг |
| Примечание | Текст | Дополнительная информация | - |

## 3. Валюты (Currencies)

| Поле | Тип | Описание | Пример |
|------|-----|----------|--------|
| Код валюты | Текст | ISO код | EUR |
| Название | Текст | Название валюты | Евро |
| Курс к базовой | Число | Курс к базовой валюте | 1.0 (если базовая) |
| Дата обновления | Дата | Дата обновления курса | 2024-01-15 |

**Базовая валюта:** EUR (по умолчанию)
