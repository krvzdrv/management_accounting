# Структура отчетов и формулы

## 1. Отчет о взаиморасчетах (Intercompany Balances)

Этот отчет показывает кто кому должен на основе всех операций.

### Структура отчета:

| От кого | Кому | Сумма в EUR | Описание |
|---------|------|-------------|----------|
| Алюминею | Версатрейд | 5000.00 | За изготовление профилей |
| Версатрейд | Алюминею | 2000.00 | Возмещение части расходов |
| Алюминею | Flexy | 3000.00 | За поставку профилей |
| Версатрейд | Посредник | 1500.00 | За доставку профилей |

### Формулы для расчета:

**Итоговая задолженность Алюминею → Версатрейд:**
```
=SUMIF(Закупки!Кто оплатил; "Версатрейд"; Закупки!Сумма в EUR) 
+ SUMIF(Расходы!Кто оплатил; "Версатрейд"; Расходы!Сумма в EUR)
- SUMIF(Движения денег!От кого; "Версатрейд"; Движения денег!Сумма в EUR)
+ SUMIF(Движения денег!Кому; "Алюминею"; Движения денег!Сумма в EUR)
```

**Итоговая задолженность Версатрейд → Алюминею:**
```
=SUMIF(Движения денег!От кого; "Версатрейд"; Движения денег!Сумма в EUR)
- SUMIF(Закупки!Кто оплатил; "Версатрейд"; Закупки!Сумма в EUR)
- SUMIF(Расходы!Кто оплатил; "Версатрейд"; Расходы!Сумма в EUR)
```

## 2. Отчет P&L (Прибыль и Убыток)

### Структура отчета:

| Статья | Сумма в EUR |
|--------|-------------|
| **Выручка (без НДС)** | =SUM(Продажи!Сумма в EUR) |
| **Себестоимость** | =SUM(Закупки!Сумма в EUR) |
| **Валовая прибыль** | =Выручка - Себестоимость |
| **Расходы** | =SUM(Расходы!Сумма в EUR) |
| **Чистая прибыль** | =Валовая прибыль - Расходы |

## 3. Остатки товаров (Inventory)

### Структура отчета:

| Товар | Приход | Расход | Остаток | Средняя себестоимость | Стоимость остатка |
|-------|--------|--------|---------|----------------------|-------------------|
| Профиль 1 | 100 | 50 | 50 | 10.50 | 525.00 |

### Формулы:

**Остаток:**
```
=SUMIF(Закупки!Товар ID; A2; Закупки!Количество) 
- SUMIF(Продажи!Товар ID; A2; Продажи!Количество)
```

**Средняя себестоимость (FIFO или средневзвешенная):**
```
=SUMPRODUCT((Закупки!Товар ID=A2)*Закупки!Количество*Закупки!Цена за единицу) 
/ SUMIF(Закупки!Товар ID; A2; Закупки!Количество)
```

**Стоимость остатка:**
```
=Остаток × Средняя себестоимость
```

## 4. Статистика расходов

### Структура отчета:

| Категория | Сумма в EUR | % от общего |
|-----------|-------------|-------------|
| Аренда | 500.00 | 25% |
| Транспорт | 300.00 | 15% |
| Итого | 2000.00 | 100% |

### Формулы:

**Сумма по категории:**
```
=SUMIF(Расходы!Категория ID; A2; Расходы!Сумма в EUR)
```

**Процент:**
```
=B2/SUM(Расходы!Сумма в EUR)*100
```

## 5. Отчет по финансированию операций

Показывает кто за что платил:

| Операция | Тип | Сумма | Кто оплатил | Статус |
|----------|-----|-------|-------------|--------|
| Закупка от Flexy | Закупка | 3000 | Версатрейд | Оплачено |
| Изготовление профилей | Расход | 5000 | Версатрейд | Частично возмещено |

### Формулы:

**Группировка по "Кто оплатил":**
```
=QUERY(Закупки!A:O; "SELECT Кто оплатил, SUM(Сумма в EUR) WHERE Кто оплатил IS NOT NULL GROUP BY Кто оплатил"; 1)
```
