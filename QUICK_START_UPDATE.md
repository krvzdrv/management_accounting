# Быстрый запуск системы обновления

## Первый запуск

Выполните в терминале:

```bash
cd "/Volumes/02 Data/work/Alumineu/GitHub/management_accounting"
node update-scripts.js
```

## Что произойдет

1. **Авторизация через OAuth:**
   - Скрипт откроет браузер или покажет URL
   - Скопируйте URL и откройте в браузере
   - Авторизуйтесь и скопируйте код
   - Вставьте код в терминал

2. **Обновление файлов:**
   - Скрипт попытается загрузить файлы из GitHub
   - Если GitHub не настроен, будет ошибка
   - В этом случае нужно модифицировать скрипт для чтения из локальной папки

## Если возникла ошибка "GitHub repo not found"

Скрипт пытается загрузить файлы из GitHub. Если репозиторий не настроен, нужно:

1. Либо настроить GitHub репозиторий в `.env`
2. Либо модифицировать скрипт для чтения из локальной папки

**Быстрое решение:** Модифицируем скрипт для чтения из локальной папки.
