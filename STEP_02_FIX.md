# ШАГ 2: Исправление установки

## Что произошло

1. Установка Homebrew была прервана (вы нажали не Enter)
2. Была опечатка: `rew` вместо `brew`

## Что сделать

### 1. Завершите установку Homebrew

Если установка Homebrew еще не завершена, выполните снова:

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

**Важно:** Нажмите **Enter/Return** когда попросит продолжить.

После установки Homebrew выполните команду, которую покажет установщик (обычно что-то вроде):
```bash
echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zshrc
eval "$(/opt/homebrew/bin/brew shellenv)"
```

### 2. Проверьте что Homebrew установлен

```bash
brew --version
```

Должна показаться версия Homebrew.

### 3. Установите Node.js

**Правильная команда (обратите внимание на `brew`, а не `rew`):**

```bash
brew install node
```

### 4. Проверьте установку

```bash
node --version
npm --version
```

Должны показаться версии Node.js и npm.

### 5. Перейдите в папку проекта

Вы сейчас в домашней директории (`~`). Перейдите в папку проекта:

```bash
cd "/Volumes/02 Data/work/Alumineu/GitHub/management_accounting"
```

Или если вы используете относительный путь:
```bash
cd "путь/к/вашему/проекту"
```

## Альтернатива: Использовать Python

Если не хотите устанавливать Node.js, проверьте Python:

```bash

```

Если Python установлен (версия 3.7+), можно использовать Python версию скрипта.
