# Исправление ошибки invalid_grant

## Проблема

Ошибка `invalid_grant` означает что:
- Код авторизации уже использован (коды одноразовые)
- Код истек (коды действительны несколько минут)
- Код неправильно скопирован

## Решение: Получить новый код

### Шаг 1: Запустите скрипт снова

```bash
cd "/Volumes/02 Data/work/Alumineu/GitHub/management_accounting"
node update-scripts.js
```

### Шаг 2: Получите новый код авторизации

1. Скрипт покажет новый URL
2. Откройте URL в браузере
3. Авторизуйтесь
4. Скопируйте **новый** код из URL
5. Вставьте в терминал **быстро** (коды истекают через несколько минут)

### Шаг 3: Важно - делайте быстро

- Коды авторизации действительны **только несколько минут**
- Скопируйте код сразу после авторизации
- Вставьте в терминал сразу же
- Не задерживайтесь между шагами

---

## Альтернатива: Удалить старый токен

Если проблема повторяется, попробуйте удалить старый токен:

```bash
cd "/Volumes/02 Data/work/Alumineu/GitHub/management_accounting"
rm -f token.json
node update-scripts.js
```

Это заставит скрипт запросить новый код авторизации.

---

## Проверка кода

Убедитесь что код скопирован полностью:
- Код должен начинаться с `4/0A...`
- Код должен быть длинным (50+ символов)
- Не должно быть пробелов в начале или конце
- Скопируйте код сразу после `code=` и до `&` (или до конца URL)
